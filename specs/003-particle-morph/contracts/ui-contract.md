# UI Contract: particle-morph（パーティクル・モーフィング）

**Date**: 2026-07-15 | **Plan**: [../plan.md](../plan.md) | **Spec**: [../spec.md](../spec.md)

本おもちゃが利用者と E2E テストに対して公開する、観測可能なふるまいの契約。
実装の内部構造（シェーダー・粒子数など）はこの契約に含まれない。

## ルートとトップページ

| 項目 | 契約 |
|------|------|
| URL | `/particle-morph` |
| トップページ一覧 | `src/toys/registry.ts` 経由で「パーティクル・モーフィング」として表示され、リンクで遷移できる（FR-001） |
| 戻る導線 | ページ内にトップページ（`/`）への戻るリンクを常に表示する（FR-015） |

## 画面要素（WebGL 利用可能時）

| 要素 | 契約 |
|------|------|
| 見出し | ページ名を示す見出し（`h1`）を持つ |
| キャンバス | `<canvas>` が主表示領域を占め、粒子の造形が動き続けている（FR-002） |
| 造形名ラベル | 現在の造形のひらがな表示名を常時表示する（FR-006）。変形完了を待たず、変形開始時に次の造形名へ切り替わる。`aria-live="polite"` で支援技術にも通知する |
| 操作ヒント | 「クリックで へんしん」等の短いヒントを常時表示する（FR-006） |

**造形の表示名と順序**（FR-005。E2E はこのラベル文字列で検証する）:

| 順序 | id | 表示名 |
|------|----|--------|
| 1 | `sphere` | きゅうたい |
| 2 | `torusKnot` | むすびめ |
| 3 | `galaxy` | ぎんが |
| 4 | `wave` | なみ |
| 5 | `text` | もじ「あっ」 |

## 操作の契約

| 操作 | 反応 |
|------|------|
| キャンバス領域のクリック / タップ | 次の造形への変形を開始し、造形名ラベルが切り替わる（FR-003） |
| Space キー | クリックと同じ（FR-003）。ページスクロールを引き起こさない |
| 変形中の再操作 | 無視される。表示は壊れず、ラベルも変わらない（FR-007） |
| ポインタを粒子群に重ねる / 離す | 周辺粒子が避け、離すと戻る（FR-008）。ラベル等の DOM には影響しない |
| 無操作 約 8 秒 | 自動で次の造形へ変形し、ラベルが切り替わる（FR-010） |
| テーマ切替（アプリ共通の ThemeToggle） | 再読み込みなしで背景・粒子の配色が 1 秒以内に切り替わる（FR-011, SC-006） |

## 環境による分岐

| 環境 | 契約 |
|------|------|
| WebGL 不可 | キャンバスの代わりに、3D 表示に対応していない旨の日本語メッセージを表示する。見出し・戻るリンクは表示され、エラー画面・白画面にしない（FR-013） |
| `prefers-reduced-motion: reduce` | 自動変形とポインタ反発を行わない。緩やかな動きのみ（FR-012）。手動変形は引き続き可能 |
| リサイズ / 端末回転 | 造形が画面内に収まり続け、ページ全体に横スクロールが発生しない（FR-014） |

## 非機能の契約（E2E で観測する範囲）

- ページ表示から 3 秒以内にキャンバスの描画が始まる（SC-001）
- 表示〜変形操作〜一巡の間、ブラウザコンソールにエラーが出ない
- 保存データを作らない（localStorage への新規キー書き込みなし）
